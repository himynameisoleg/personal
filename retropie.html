Retro Pie

Checklist:

  • Raspberry Pi 2 Model B
  • Micro USB Charger (5 Volt)
  • HDMI Monitor
  • 8gb MicroSD (or larger)
  • Keyboard + Mouse
  • Ethernet Cable
  • USB WiFi Adapter Optional
  • USB Bluetooth Adapter Optional
  • Enclosure or Case Optional
  • USB gamepad (we’ll be using a PS4 controller wired)

This tutorial will be the easiest / fastest way to setup RetroPie for instant retro gaming!
We’ll be using a Mac for this installation.

Software Needed:

  • RetroPie Image
  • Etcher

Preparing the Pi

This part is pretty straight forward.
Plug in display, mouse, keyboard, WiFi adapter (or ethernet cable), and controller.

Preparing the SD

Very easy as well.
Insert the MicroSD into your Mac using an adapter or hub.
Open up Etcher.


Select the RetroPi .img file we downloaded.
Etcher should automatically detect your SD card.
If it doesn’t let you, then open your DiskUtility and reformat the card with default settings.
You can name it whatever you like.


Go back to Etcher, hit Flash! and wait a few minutes while the image is flashed onto your SD.
You’re done! Next we are going to configure controllers and add some games.
Insert the SD into your Raspberry Pi and plug in the Micro USB for power.
If everything was done correctly you should soon see a splash screen with the RetroPie logo.

In Part 2 we will add games to your newly setup Pi.
